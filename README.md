# Aerolab

WEB:

- Si es la primera instalar dependencias con:

```bash
    npm install
```

- Agregar la URL de la API en el .env en la siguiente variable:

  - REACT_APP_API_URL: (por ejemplo: http://localhost:8080)

- Para ejecutar usar:

```bash
    npm start
```

API:

- Si es la primera instalar dependencias con:

```bash
    npm install
```

- Para ejecutar usar:

```bash
    npm run dev
```
